# Changelog - Post Backup & Restore Pro

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-06-25
### Added
- Initial Stable Release of **Post Backup & Restore Pro**.
- Enterprise-grade OOP PHP architecture matching WordPress Core formatting guidelines.
- Autoloader mapping for PSR-4 class matching.
- High-integrity ZIP compression engine supporting standard directories (`/posts/`, `/images/`, `/metadata/`).
- XML, CSV, and JSON schema data export files detailing posts attributes.
- Secure, robust sideload mechanisms for internal and external inline images.
- Comprehensive security shielding against directory traversal attacks (Zip Slip).
- SEO support mapping for Yoast SEO, RankMath SEO, and SEOPress parameters.
- Built-in live logging panel tracking system activity.
- Complete responsive WP Admin template designs.
